# 📊 EDA-YouTube-Usage-Data-Set-(Python)

## 📌 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** on YouTube data to uncover meaningful insights about trending videos, audience engagement, and content performance.

The analysis includes data cleaning, preprocessing, visualization, and extracting insights using Python libraries like Numpy, pandas, Matplotlib.

---

## 🚀 Objectives
- Analyze YouTube trending data
- Understand audience engagement patterns
- Find the most popular video categories
- Visualize trends and correlations
- Generate actionable insights from data

---

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

---

## 📂 Dataset Features
The dataset contains:
- Video Title
- Channel Name
- Views
- Likes
- Comments
- Category
- Publish Time
- Trending Date
- Tags

---

## 📊 Analysis Performed

### 🔹 Data Cleaning
- Handling missing values
- Removing duplicates
- Data type conversion

### 🔹 Exploratory Data Analysis
- Most trending categories
- Top channels by views
- Views vs Likes analysis
- Comment analysis
- Correlation heatmaps
- Engagement analysis

### 🔹 Data Visualization
- Bar Charts
- Pie Charts
- Heatmaps
- Scatter Plots
- Interactive Plotly Visualizations

---

## 📈 Key Insights
- Entertainment and Music categories receive high engagement.
- Strong correlation exists between views, likes, and comments.
- Trending videos show specific upload timing patterns.
- Higher engagement often leads to increased visibility.
